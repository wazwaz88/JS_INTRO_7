Function: it is a block of code could be reused with a function name. 

It is useful to create a logic and implement it in a function and reuse it many times.
getRandomNumber(5, 15);
getRandomNumber(1, 100);


THINGS TO DISCUSS WHILE CREATING A FUNCTION?
TASK: what your function MUST do
ARGUMENTS: Does it take any argument? How many? What function should do with these arguments?
RETURN: Should it return it or not?
